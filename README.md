## 📐 Quy ước chung

### Cấu trúc thư mục

- Mỗi bài tập nằm trong thư mục riêng: `HomeworkXX/`
- Trong mỗi bài tập, mỗi sinh viên có thư mục riêng theo định dạng: `(MSSV)-(HọTên)/`

### Đặt tên file báo cáo

File báo cáo PDF phải đặt tên theo định dạng:

```
(MSSV)_(HọTên)_W(tuần số).pdf
```

Ví dụ: `20235658_VuDucHoangAnh_w1.pdf`

### Git

- Mỗi sinh viên làm việc trên **branch riêng**, đặt tên theo MSSV: `hw(week)/(MSSV)-(NAME)`
- Merge vào `main` sau khi hoàn thành, giải quyết conflict trước khi merge
- Không push trực tiếp lên `main`

### Khác
- Build tool: **Maven (nếu cần)**
- IDE: **IntelliJ IDEA**
- Terminal: **PowerShell/Git Bash/Mac Zsh**
